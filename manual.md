# Manual

## Main window

### Mouse
- Scroll: Move
- Shift+Scroll: Zoom
- Pinch in/out: Zoom
- Double click: Reset move/zoom

### Shortcuts
- space: Play/Pause
- ⌘+A: Switch auto mode ON/OFF
- ⌘+R: Randomize attractions
- ⌘+P: Show parameters
- ⌘+S: Show statistics

## Contol window

### Attraction

- Left/Right click value: ±0.1
- Shift+Left/Right click value: Set 1/-1
